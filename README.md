# DA-Exercises
Data Analytics Exercises 

This repository contains a collection of SQL exercises and practical tasks completed as part of my data analytics learning journey.
The exercises cover SQL fundamentals, advanced SQL techniques, and applied analytics using different tools such as Snowflake, Microsoft SQL Server, and Google BigQuery.

### Contents
1. Research Assignment

A written research task exploring key data analytics concepts and methodologies.

2. SQL Exercises
 
 - Exercise 1 – SQL Fundamentals - SELECT statements, filtering, sorting, and basic query structure.
 - Exercise 2 – SQL Aggregate Functions & Operators - Using COUNT, SUM, AVG, MIN, MAX and logical operators.
 - Exercise 3 – SQL Case Statements - Conditional logic in SQL queries.
 - Exercise 4 – SQL JOINS - INNER, LEFT, RIGHT, and FULL joins.
 - Exercise 5 – Advanced SQL (Date Functions) - DATEADD, DATEDIFF, EXTRACT, formatting dates.
 - Exercise 6 – Advanced SQL (NULL Functions) - Handling missing values using COALESCE, NULLIF, ISNULL, etc.
 - Exercise 7 – SQL Union & Union ALL - Combining data across multiple tables.

3. Practical SQL Exercises

- Practical – SQL Fundamentals (Snowflake) - Queries applied on Snowflake environment.
- Practical – SQL Joins
- Practical – Advanced SQL (NULL Functions)
- Practical – Soccer SQL Questions (MS SQL Server) - Sports dataset analytics using SQL Server.
- Practical – Retail Sales (Google BigQuery) - Retail insights derived using BigQuery SQL.

### Tools & Technologies Used for SQL

- Snowflake
- Microsoft SQL Server
- Google BigQuery
- Databricks
